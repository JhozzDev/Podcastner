
# Podcastner

Desktop app en C# con podcasts en variedad de idiomas para acquirir idiomas mendiante tecnicas de shadowing y reforzar listening.

## Contiene

- +100 Podcast
- Subtitulos
- Busqueda con TaddyAPI
- Guardado de podcast
- Dictionario de palabras desconocidas

## Tecnologias
- .NET (WPF)
- Taddy Podcast(API)
- C# (.NET)
